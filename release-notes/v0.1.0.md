## Overview

OpenAuthSter Issuer is a production-ready authentication server that enables you to host multiple projects with independent authentication configurations on a single Cloudflare Workers deployment. Built for the edge, it delivers low-latency authentication globally.

## âœ¨ Key Features

### Multi-Tenant Architecture

- **Project Isolation:** Each project maintains its own authentication configuration, users, and sessions
- **Centralized Management:** Single deployment manages multiple applications
- **Invite System:** Control project access with invite links
- **Template Copy:** Create Custom i18n for the login/register flow

### Authentication Methods

#### Email Authentication

- **Password-based Login:** Traditional email/password authentication
- **Magic Links:** Passwordless authentication via email codes
- **Custom Email Provider:** Integrate your own email service
- **Resend Integration:** Built-in support for Resend email service
- **Customizable Templates:** Personalized email templates per project

## ðŸš€ Getting Started

### Prerequisites

- Cloudflare account with Workers and D1 access
- Wrangler CLI installed
- Bun.js or Node.js

### Quick Start

```bash
# Install dependencies
bun install

# Create D1 database
wrangler d1 create openauth-db

# Configure wrangler.json with your database credentials

# Run migrations
wrangler d1 migrations apply AUTH_DB --remote

# Deploy to Cloudflare Workers
wrangler deploy

# Or develop locally
wrangler dev --port 8788
```

For detailed setup instructions, see the [README](../README.md).
